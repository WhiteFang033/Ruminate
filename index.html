User Manual
1. Welcome! This is a notes app called "Ruminate". It utilizes the Local Storage of browser to store data since Local Storage of a browser withstand page reload and even browser restart. So data won't be lost unless Local Storage itself is cleared or browser is uninstalled.

2. This is a sample note engraved (written) in the original HTML document. It can be deleted or edited but if Local Storage is affected, this card may appear again.

3. There are other like this sample note, namely sample task and a sample recycled note. Both of them are written into the original HTML document and I think you figured out what that means.

4. Each Note offers 3 options. First is background change option, clicking on the round images will change the background. Second option is the Delete Option, clicking this will delete the respective note from notes section but you can later recycle it from Recycle Bin, in case you change your mind or your mouse might have mis-clicked. The last option is close button, this does nothing but closes the respective note, that's all.

5. That's all for now I think. Good Luck using this app!
